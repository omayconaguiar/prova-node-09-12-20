## Setup Para rodar local

```js
yarn install
```

### Migrations

```js
yarn migrate:latest
```

```js
yarn seed
```

### Run 

```js
yarn dev
```

### Test

```js
yarn test
```

### Lint

```js
yarn lint
```

### Autenticação

Abra o arquivo .env.sample e veja a api-key que será usada nas headers das requisições.
